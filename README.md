# Documentación de Markdown

Este repositorio contiene una serie de ejemplos y comandos básicos de Markdown, un lenguaje de marcado ligero utilizado para crear documentación de manera fácil y rápida.

## Encabezados

Los encabezados se crean utilizando el símbolo `#` seguido de un espacio. El número de `#` indica el nivel del encabezado.

# Encabezado de Nivel 1
## Encabezado de Nivel 2
### Encabezado de Nivel 3

## Listas

### Listas no ordenadas

Se usan guiones `-`, asteriscos `*` o signos más `+` para crear listas.

- Primer ítem
- Segundo ítem

### Listas ordenadas

Las listas numeradas se crean utilizando números seguidos de un punto.

1. Primer ítem
2. Segundo ítem

## Formato de texto

### Negrita

Para poner texto en negrita, se usan dos asteriscos `**` o guiones bajos `__`.

**Texto en negrita**

### Cursiva

La cursiva se logra con un solo asterisco `*` o guion bajo `_`.

*Texto en cursiva*

## Enlaces

Puedes crear enlaces de la siguiente manera:

[GitHub](https://github.com)

## Imágenes

Las imágenes se insertan con un signo de exclamación `!` seguido de un enlace.

![Texto alternativo](https://ruta-a-la-imagen.com)

## Citas

Las citas se generan utilizando el símbolo `>`.

> Esto es una cita.

## Código

### Código en línea

Se utiliza la tilde invertida ` para envolver el código.

`print("Hola Mundo")`

### Bloques de código

Se usa tres tildes invertidas ``` para bloques de código.
