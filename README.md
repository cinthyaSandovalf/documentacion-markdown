# Documentación de Markdown

Este repositorio contiene una serie de ejemplos y comandos básicos de Markdown, un lenguaje de marcado ligero utilizado para crear documentación de manera fácil y rápida.

## Encabezados

Los encabezados se crean utilizando el símbolo `#` seguido de un espacio. El número de `#` indica el nivel del encabezado.

```markdown
# Encabezado de Nivel 1
## Encabezado de Nivel 2
### Encabezado de Nivel 3
